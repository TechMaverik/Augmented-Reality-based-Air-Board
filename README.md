# Augmented Reality Based Airboard Project
## Installation
1. Install the latest version of python3
2. Clone the repository https://github.com/TechMaverik/Augmented-Reality-based-Air-Board.git
3. Go to the repository directory
4. open terminal
5. Create a virtual environment by using the command <code> python3 -m venv .venv </code>
6. Install the required libraries by using the command <code>pip3 install -r requirements.txt</code>

## Running the application
1. Activate the virtual environment in teminal by using the command <code>source .venv/bin/activate</code>
2. <code>python3 main.py</code>
3. ENJOY
